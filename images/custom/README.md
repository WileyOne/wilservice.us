# Custom Branding Assets

Samples for wilservice.us favicons and branding. Based on award-winning small business tech branding research: minimalist, geometric, high-contrast, readable at favicon scale.

## Brand Context (from site)

- **Services:** Repair, training, networks, computer security, accounts & sign-in, personal security
- **Why us:** No long-term contract, local, plain language, built for your size, security and privacy
- **Colors:** Teal accent #22b4b8, dark bg #0c0f14

## 5 Alternate Design Themes

| Folder | Theme | Description |
|--------|-------|-------------|
| `1/` | **Minimal Geometric** | Teal on dark, clean lines, current style |
| `2/` | **Tech/Cyber** | Circuit-inspired, electric blue glow, digital aesthetic |
| `3/` | **Human-Centered** | Production colors (teal #22b4b8, dark #0c0f14). Trust, empathy, strength, protection. WIL|SERVICE wordmark with S split to join them. MC Escher–inspired tessellation/impossible figures. |
| `4/` | **Security-Focused** | Bold shield, navy blue, protection and trust |
| `5/` | **Professional/Corporate** | Classic refined typography, navy + teal, established |
| `6/` | **Connection/Cord** | Network cord motif, v tail as cable, S joins WIL and ERVICE. Human-centered. |
| `7/` | **Trust/Hand** | Handshake, human trust, protective. WIL/S/ERVICE split, v tail as cord. |
| `8/` | **Empathy/Support** | Supportive embrace, care, lift. Human-focused. |
| `9/` | **Strength/Shield** | Shield with interwoven bands, WIL\|S\|ERVICE split, R tail as cord flowing into US three-prong plug that forms the E. Teal + charcoal. |
| `10/` | **Partnership/Circle** | Collaborative community, circular connection. |

All sets 6–10 use production colors (teal #22b4b8, dark #0c0f14), font from set 1, WIL|S|ERVICE color separation with S as transition, and v tail as network cord (not arrow).

Each folder contains the full set: favicon-w-mark, favicon-shield, favicon-network-nodes, logo-mark-dark, logo-mark-light.

## Files (per set)

| File | Use |
|------|-----|
| `favicon-w-mark.png` | Stylized "W" mark. Ties to wilservice, works at small sizes. |
| `favicon-network-nodes.png` | Abstract connected nodes. Suggests network, repair, support. |
| `favicon-shield.png` | Shield outline. Suggests security (computer + personal). |

## Usage

1. **Favicon:** Resize chosen favicon to 32x32 and 16x16. Add to site root or reference in HTML:
   ```html
   <link rel="icon" href="/images/custom/favicon-w-mark.png" sizes="32x32" />
   ```

2. **Apple touch icon:** Use 180x180 for iOS home screen.

3. **Replace samples:** These are drafts. A designer can refine or you can swap in final assets.
